# SplitPay

Shared expense tracker PWA with Google Drive receipt storage and AI scanning.

## Live App
https://neilson2005.github.io/splitpay

## Features
- Sign in with Google
- Add members and expenses
- Scan receipts with AI (Claude)
- Store receipt photos in your Google Drive
- Auto-split expenses equally
- Settle up calculator
- Works offline (PWA)
- Installable on iOS and Android home screen

## Setup
1. Open https://neilson2005.github.io/splitpay
2. Sign in with Google
3. Add members → scan receipts or add manually
4. Share the URL with friends so they can join
